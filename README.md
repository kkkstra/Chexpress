# 毕业快递价格对比

一个离线可运行的静态单页，用于对比顺丰、京东、德邦在不同目的地和重量下的毕业寄件预估总运费。

## 功能

- 按省份和地级市选择目的地，自动匹配三家快递的价目表条目。
- 图表展示 `0.5-80 kg` 或 `1-160 斤` 的重量区间；输入超过 80 kg 时，价格继续按计费重量计算，图表不再显示焦点。
- 点击重量输入框右侧单位按钮可在千克和斤之间切换。
- 用 SVG 折线图展示顺丰、京东、德邦的总运费曲线。
- 鼠标悬停图表时按当前 x 轴最近重量更新价格；点击图表可固定重量，再次点击可取消固定。
- “价格表”按钮可查看三家快递录入后的原始价格表。
- 首次打开页面会弹出价格免责声明，确认后使用 `localStorage` 记录，后续打开不再弹出。

## 计费规则

- 页面统一以斤作为内部计算单位，`1 kg = 2 斤`；输入重量按当前单位向上取到最近计费粒度，千克为 `0.5 kg`，斤为 `1 斤`。
- 顺丰：按价目表中的首重 40 斤、续重 1（40-120 斤）、续重 2（120 斤以上）累计计算。
- 京东：按价目表中的首重 30 斤、续重 1（30-120 斤）、续重 2（120 斤以上）累计计算。
- 德邦：20 斤以内按首重价；20-60 斤按续重 1；60 斤及以上按表中 60 斤价格和续重 2 计算。

## 本地运行

无需安装依赖或启动构建工具，直接用浏览器打开：

```text
index.html
```

项目文件：

- `index.html`：页面结构和弹窗。
- `styles.css`：响应式布局和视觉样式。
- `app.js`：交互逻辑、匹配规则、图表渲染和价格计算。
- `data.js`：快递价格、地区列表和数据辅助函数。

## 校验

可用 Node.js 检查脚本语法：

```bash
node --check app.js
node --check data.js
```

改动数据时建议同时人工核对来源图片，并抽查每家公司至少一个首重、续重 1、续重 2 区间的手算结果。
